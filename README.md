## 🌟 Overview

The **IHM Healthcare Foundation** (Intercontinental Healthcare & Medical Charity) is a non-profit organization dedicated to bridging the gap between compassion and action. We focus on funding healthcare initiatives across Africa, providing support for the poorest communities through innovation and community-driven support.

This repository contains the official web application for the foundation, built with a modern, high-performance tech stack.

## 🚀 Features

- **📱 Fully Responsive Design**: Optimized for all devices using Tailwind CSS 4.
- **✨ Fluid Animations**: Smooth transitions and interactive elements powered by Framer Motion.
- **💰 Secure Giving**: Multiple donation methods including PayPal, Direct Bank Transfer, and QR Scan.
- **📰 News & Events**: Stay updated with the latest foundation activities and success stories.
- **🤝 Volunteer Portal**: Streamlined process for individuals to join our mission.
- **🚑 Programs**: Detailed information on our healthcare initiatives and funding areas.
- **🍪 Privacy Focused**: Integrated cookie consent and privacy policies.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/) (Framer Motion)

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 18.x or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-repo/ihmhealthcare.git
   cd ihmhealthcare
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env.local` file in the root directory and add your API keys:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

### Development

Run the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

### Build

Create a production build:
```bash
npm run build
```
